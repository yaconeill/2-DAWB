Ejercicio 3

3.- Las variables "max-age", "path" y "domain" son nombres especiales de variables que son reconocidas por un navegador para controlar diferentes aspectos. Investiga que significa cada una de esas variables y explica su significado y uso en el propio fichero.

  • Max-age: Es el número de segundos hasta que la cookies expire. Se establece con uno o más digitos de 1 a 9. Si max-age y expire estan definidos, max-age prevalece.

  • Path: Indica una ruta URL que debe existir en el recurso de petición antes de enviar la cabecera de la cookie.

  • Domain: Especifica los host a los que la cookie será enviada, si no se especifica, se establece de forma predeterminada en la parte del host de la ubicación del documento actual, pero no incluye el subdominio.

Ejercicio 3

Realizar una versión simplificada del juego "Ahora Caigo" de TV consistente en: que al usuario le aparecerá una pregunta por pantalla. También le aparecerá la respuesta y algunas letras estarán substituidas por asteriscos (por ejemplo, a la pregunta: "Archipiélago conocido como 'Las afortunadas'" se le presentaría una respuesta (que no es única, depende de tu criterio, a más *, mayor dificultad): I**AS **N*R*AS El contenido de los asteriscos será editable, o sea, seleccionando el área con el ratón se podrá escribir y eliminar el texto de ese área. Una vez esté completada la respuesta, se pinchará en el botón de “Comprobar”. El script comprobará si está la respuesta es correcta o no, avisando del resultado. Este script irá guardando un histórico de todos los intentos, de forma que se mostrarán, en la misma pantalla, los textos que fueron enviados a comprobación. Para almacenar estos textos utilizaremos localStorage. Una vez terminado el script, deberemos hacer varias pruebas, unas poniendo bien la respuesta y otras fallando. Cuando tengamos el historial bastante lleno, cerraremos el navegador. Una vez cerrado lo volveremos a abrir y comprobaremos que se mantiene el historial. También se incorporará un botón de “Reiniciar” para eliminar nuestro historial.
