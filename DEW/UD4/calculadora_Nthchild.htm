U04T01 - Calculador visual de nth-child

Introducción

La pseudo-clase CSS :nth-child selecciona los elementos hermanos dentro del árbol del DOM, que coincidan con la condicion an+b-1, siendo n un numero natural, y teniendo un elemento padre.

De esta manera, podemos seleccionar al enésimo hijo de una caja padre utilizando el selector :nth-child . Por ejemplo: dada una lista <ul> de 10 artículos,  li:nth-child(6) selecciona el sexto artículo de la lista.

A su vez podemos utilizar fórmulas, por ejemplo li:nth-child(2n) selecciona todos los hijos pares, mientras que li:nth-child(3n) selecciona cada hijo cuyo índice es un múltiplo de 3.

En esta actividad se desarrollará una Web (utilizando JS nativo) que permita de manera visual probar los elementos afectados por una pseudo clase en la que se tiene una fórmula nth-child. 

Descripción

Inicialmente, la web tendrá un único elemento div cuyo id se llame container. A partir de este div se generarán mediante JS el resto elementos de la Web.

También se puede disponer de un fichero CSS cuyo contenido dependerá del alumnado. 

La Web tendrá 2 partes diferenciadas:

  • Opciones: 
    • Fórmula: cuadro de texto para que el usuario escriba la fórmula nth-child
    • Alto: número de celdas que hay en sentido vertical que puede ser introducir por el usuario mediante una caja de texto.
    • Ancho: número de celdas que hay en sentido horizontal. También puede ser introducida por el usuario mediante texto.
    • Color principal: se recogerá en formato hexadecimal (también se puede utilizar el input HTML5 asociado). Es el color por defecto
    • Color secundario: es el color que se utilizará para las celdas que coincidan con las fórmula nth-child
    • Aceptar: botón que realiza la acción de colorear el tablero indicado
    • Reset: botón que deja el tablero a su situación de partida
  • Tablero:
    • Se dibujarán las celdas de acuerdo con la fórmula indicada y los colores

Situación de partida

Las opciones se le mostrarán al usuario al cargarse la Web.

El tablero se cargará inicialmente con un tablero de 20 x 20. Los colores principal y secundario deben ser diferentes entre sí.

A entregar

Fichero .Zip (no RAR, ni 7zip) con los ficheros que resuelven el enunciado.

En el caso de incluir una funcionalidad adicional, añadir un fichero .txt explicativo describiendo la nueva función implementada.

El nombre del fichero debe ser U04T03_2DAW_Grupo_Nombre_Apellidos (Grupo: A o B, Nombre y Apellidos, los del alumnado),
