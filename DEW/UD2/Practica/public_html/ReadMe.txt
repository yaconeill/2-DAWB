Aportaciones al código solicitado:

Se ha añadido un set de botones y casillas de verificación:
 - Botón Reiniciar:
   El cual reinicia los valor para poder realizar una nueva ejecución, también sirve para parar una carrera en marcha.
 - Botón Animado:
   El cual realiza una animación de la carrera en una sola línea (se fuerza la activación de la casilla En una línea).
 - Casilla En una línea:
   Muestra la ejecución de paso a paso en una línea cuando está marcada.
 - Casilla Con imágenes:
   Muestra la carrera utilizando imágenes en lugar de caracteres.

Una vez se comience cualquiera de las ejecuciones, se deshabilitan todos los botones y casillas excepto el botón reiniciar, para evitar que comience una nueva ejecución sin que haya terminado la anterior.
